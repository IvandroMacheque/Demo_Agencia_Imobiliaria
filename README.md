# Sistema de Gestão Imobiliária

## Descrição

Sistema desenvolvido para auxiliar na gestão de apartamentos e controle de itens dentro de uma agência imobiliária.

O objetivo é organizar a ocupação dos apartamentos, controlar o inventário e registrar movimentações de itens entre unidades.

---

## Funcionalidades

- Cadastro de apartamentos
- Separação entre apartamentos ocupados e disponíveis
- Controle de itens por apartamento
- Registro de movimentação de itens
- Atualização automática do inventário após movimentações
- Visualização geral do status dos apartamentos

---

## Tecnologias Utilizadas

- Python
- Flet (interface)
- Estrutura baseada em lógica modular
- Organização por funções e separação de responsabilidades

---

## Estrutura do Projeto

O projeto foi organizado de forma a separar:

- Interface (telas e componentes)
- Lógica de negócio
- Manipulação de dados
- Funções auxiliares

Essa organização facilita manutenção e futuras melhorias.
